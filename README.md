# Biped Robot Simulation (ROS2)

## Overview
A robotics simulation project focused on modeling and controlling a biped robot using ROS2 and Gazebo. The project explores control logic, sensor messaging, and simulation workflows.

## Features
- ROS2-based architecture
- Gazebo simulation environment
- Python-based control logic
- Modular and extensible design

## Technologies Used
- Python
- ROS2
- Gazebo

## Motivation
This project was created to gain hands-on experience with robotics middleware and simulation, with applications in control systems and autonomous robotics.

## Current Status
- Biped simulation running in Gazebo
- Control and messaging pipelines implemented

## Planned Improvements
- Migration to Isaac Sim
- Advanced control strategies
- Improved physics and sensor realism
